## 授業でのLaravelの扱いと、このリポジトリについて

これは、授業で"laravel-test"として扱っているプロジェクトの最新状態を反映したものです。
コミットの際のメッセージは、いまのところ以下のような規則になっています：

- commit-日付
- test

このうち**commit-日付**というものが、メインとなる更新です。授業に参加できなかった場合などは、この日付情報を参考にしてください。
